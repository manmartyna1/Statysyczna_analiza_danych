# Data Analysis Projects / Statystyczna analiza danych

This repository contains selected data analysis projects completed in Python using Google Colab / Jupyter Notebook.

## Contents

### Main project
- **housing-price-regression-analysis.ipynb**  
  A regression analysis project based on the Boston Housing dataset, including data preprocessing, exploratory analysis, visualization, and model evaluation.

### Additional exercises
- **chocolate-data-cleaning.ipynb**  
  A smaller data cleaning and preprocessing exercise based on a chocolate dataset.

## Topics covered
- data cleaning and preprocessing
- exploratory data analysis (EDA)
- statistical analysis
- data visualization
- regression modeling

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- scikit-learn
- Google Colab / Jupyter Notebook
